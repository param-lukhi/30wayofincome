What is TaskRabbit?

TaskRabbit ek platform hai jo clients aur workers (Taskers) ko connect karta hai. Clients local tasks post karte hain aur Taskers un tasks ko complete karke paisa kamate hain.

  • Furniture assembly

How TaskRabbit Works? (Step-by-Step)

  1. TaskRabbit par free account create karein
  2. Profile aur skills add karein
  3. Hourly rate set karein
  4. Nearby tasks receive karein
  5. Client aapko book karta hai
  6. Task complete karein
  7. Payment direct bank me milti hai

How to Create TaskRabbit Account

  • Visit: www.taskrabbit.com
  • Click on Become a Tasker
  • Email aur phone number se signup karein
  • Identity verification complete karein

  • Profile approval ka wait karein
